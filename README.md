## 搬瓦工优惠码使用教程：手把手教你填对位置

这部分是全文重点。很多人搜"搬瓦工优惠码使用教程"卡住，不是不会填，是**找不到填的地方**。搬瓦工的结账流程对新手不算友好，那个输入框藏得有点深。

### 购买时使用优惠码（最常见场景）

1.  **选套餐**：进入搬瓦工官网，挑一个你想要的套餐，点击 Order Now 进入配置页。
2.  **选配置**：在配置页选择计费周期（Billing Cycle，比如月付 / 季付 / 年付）和机房（Location），点击 Continue 进入购物车。
3.  **找到输入框**：在购物车结算页（Cart 页），向下滚动，你会看到一个写着 **"Promotional Code"** 的输入框——它通常在订单摘要下方，位置不太显眼，别一眼扫过去就点了 Checkout。
4.  **填码 + 验证**：把优惠码（比如 `NODESEEK2026`）粘进去，**先点旁边的 "Validate Code" 按钮**，不要直接点结账。验证成功后，页面会刷新，订单总额旁边会出现一行绿色的折扣金额。
5.  **确认后结账**：看到折扣生效了，再点 Complete Checkout 走支付流程。搬瓦工支持支付宝、PayPal、信用卡，国内用户用支付宝最方便。

**关键动作就一个**：先 Validate，再 Checkout。顺序反了折扣不会自动叠加。

### 续费时使用优惠码

好消息：续费不用你再填一遍。

搬瓦工的循环折扣机制会在你续费时**自动套用购买时用过的那个优惠码**，系统直接算好折后价，你只要付款就行。目前后台不支持续费时新增或更换优惠码——所以**第一次下单时填对码特别重要**，这决定了你之后每年续费能省多少。

### 升级套餐时更换优惠码

如果你买的是低配套餐，后面想升级到更高配置，搬瓦工是支持在升级时**换一个新的优惠码**的。操作路径是：登录后台 → Services → My Services → 选择要升级的 VPS → Upgrade/Downgrade，在升级结算页同样会有 Promotional Code 输入框，这时可以填一个力度更大的新码。换句话说，**升级是一次"重新选码"的机会**，别浪费。
